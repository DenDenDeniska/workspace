Very Important Notes:

  • The subject of the e-mail message should begin with the program name, e.g. "SiSoft Sandra" followed by the type (Standard or Professional) and version (see Help-About), e.g. "Sisoft Sandra Standard Version 97.1.1.0 : Program Errors". E-mail is sometimes re-routed via different accounts and it may get lost otherwise...
  • We reply to almost all e-mail submissions and inquiries. If you do not receive a reply within a reasonable time, send your message again, giving a better return path (a surprising number of mailers do not insert a correct From: address!)
  • We reserve the right to delete any derogatory/insulting/illegal/etc. messages and add the address of the sender to the killfile. This means that any such messages will not be downloaded by our mailer in the future. Criticism is welcome, insults are not.
  • Please try to include a report (Use the Create a Report Wizard to save one to disk or e-mail it - choose Text Format) when you have a hardware problem. Otherwise we cannot help. Very important! (ask yourself - can a doctor diagnose me without even seeing me? we cannot read minds)
  • Please write in English.
