Installing the Program

Please run Setup.exe and follow the instructions.

Un-Installing/De-Installing the Program

Go to Control Panel - Add/Remove Programs and choose SiSoft Sandra. Click on Add/Remove button and follow the instructions. If you did not like this release maybe you'd care to try other versions. Thanks.

If any of the above files are missing, try to download the package from another site. If some are corrupted - there must have been an error during transmission (check that you downloaded in binary mode) - repeat the operation.

All program files are compressed, so you should let the Install Wizard decompress & copy them to the hard disk. The Install program will also create a program group and add the program items. The InstallShield is the installer for Windows applications and is used by most commercial and shareware applications. It is arguably the best so you are very likely to have it already installed on your machine.

Making a 1.44MB Floppy Disk Install

While the total size of the ZIP is more than 1.44MB, the files have been designed to fit on two (or more) floppy disks. To create a 1.44MB floppy disk install do the following:

  • Disk 2 - Copy file data2.cab only on this disk.
  • Disk 1 - Copy all other files on this disk. They should fit

That's it! Unfortunately, you cannot make an install using smaller floppy disks. If this is a problem, contact us.
