Who/What is Sandra?

SiSoft Sandra (the System ANalyser, Diagnostic and Reporting Assistant) is an information & diagnostic utility. It should provide most of the information (including undocumented) you need to know about your hardware, software and other devices whether hardware or software.

Sandra provides similar level of information to Norton SI, Quarterdeck WinProbe/Manifest, etc. It is a native 32-bit Win32 application. It also provides information on 16-bit Windows and DOS unlike many other system information programs...

However, if you want DOS information, then a DOS system analyser will be far better simply because it runs under DOS so it can do everything it wants unlike Windows applications, especially 32-bit ones...

Also, for 16-bit Windows sub-system information, a 16-bit information/diagnostic utility might be better, e.g. you may find SiSoft SAW quite useful as long as you have 16-bit Windows apps. Actually, Windows 95/98 are mostly 16-bit (for good and bad reasons), so until you move over to NT/2000, 16-bit things are here to stay.

General Sandra Information

There are two versions of Sandra, in line with industry standards:

  • Sandra Standard (free for home use only - no nag screens, time limit, etc.)
  • Sandra Professional (commercial)

SiSoft Sandra Standard Information

  • 50 Information, Benchmarking, Listing & Testing/Diagnostic Modules.
  • Multi-threaded Wizards for all tasks (e.g. Create various reports).
  • Multi-processor designed (multi-threaded) benchmarks - up to 32 CPUs.
  • Full 32-bit, all-new code. Uses all Windows 95/98/Me (& 2000) extensions.
  • You still get information on the 16-bit subsystem.
  • Plug-and-Play support.
  • DMI/SMBIOS support.
  • VESA, VBE, ACPI, APM, BIOS SMB, I2C & SMBus support.
  • Supports AGP, PCI, VESA & EISA.
  • WinG, DirectX (DirectDraw, Direct 3D, DSound, DMusic, DPlay, DInput), OpenGL, Glide support and information.
  • Detects all new CPUs including the Intel Pentium MMX/Pro/II/III/4, Celeron, Xeon /III/4, OverDrive; Cyrix 6x86, 6x86MX, MII, MIII, MediaGX, MediaGXm; AMD K5, K6, K6-2/III, Athlon/Duron; NexGen 586, 686; IDT WinChip C6, WinChip 2, WinChip 2-3D, VIA Cyrix III, etc.
  • Detects MMX, MMX Enhanced, 3DNow!, 3DNow! Enhanced, SSE(2) and other CPU information like cache, speed, etc.
  • Advanced settings information on modern CPUs.
  • Detects Intel, AMD, ALi, SiS, VIA and all other chipsets on the market.
  • Network & Internet information.
  • Can save, print, fax, e-mail and upload a report.
  • Can export a SMS report for Microsoft SMS management system.
  • Can export a RPT (hierarchical) report that can be imported into most programs.
  • Can export a XML report for electronic data interchange.
  • Can export a HTML report directly (with CSS support) or display one dynamically from the XML using XSL & CSS.
  • InstallShield Install/UnInstall wizard.
  • Compatible with Windows 98's Task Scheduler or the Sage interface of the Windows 95 Plus! pack.
  • Looks better in HiColour/TrueColour (16-bit+) modes.
  • See Compatibility List for more information.

SiSoft Sandra Professional Information

  • Same capabilities as the Standard Version, Plus:
  • 70 total, i.e. 20 more modules. (these are present in the Standard version, but don't run)
  • Network support (IPX/SPX, TCP/IP).
  • See Compatibility List for more information.
