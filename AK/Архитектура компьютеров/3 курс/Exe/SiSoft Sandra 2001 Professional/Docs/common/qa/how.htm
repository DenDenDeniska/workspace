SiSoft Sandra - The Diagnostic Tool, Q & A - How do I?

This document provides some frequently asked questions about SiSoft Sandra. Please read the Help File as well!

Q: How do I get the latest version of Sandra?
A: Click here for more information.

Q: How do I optimise the system using the Performance Tune-Up wizard?
A: Click F1 in the module in question and follow the instructions in the help file. You will need to refer to the help topic of the actual module since the wizard does not have a database of the tips.

Q: How do I update my system's BIOS?
A: If Sandra tells you to update your system's BIOS then it's advisable to do it if you want to keep up-to-date. However, if everything is working properly you should not do it - don't fix it if it ain't broken.

Computer Type What to do
386, 486 with PROM BIOSContact Unicore Software or Micro Firmware for more information. You will need to pay for the update.
586, Pentium and later with FLASH BIOS Check the Web page of the mainboard's manufacturer. If you do not know it, try Wim's Flash BIOS List. The update is FREE.
Other computers with no-name board manufacturer or no BIOS update available. Contact Unicore Software or Micro Firmware for more information. You will need to pay for the update. Or try using other manufacturer's BIOS (for the same chipset) as a last resort.

Do note that we take NO RESPONSIBILITY for what might happen. Read very carefully the install procedure and back-up your BIOS before proceeding. Disable BIOS caching and shadowing and run the flasher from a CLEAN DOS prompt not a Windows's shell. Better use a floppy disk as boot-up.
