SiSoft Sandra - The Diagnostic Tool, Q & A - Registering

This document provides some frequently asked questions about SiSoft Sandra. Please read the Help File as well!

Q: How do I register/purchase Sandra Professional?
A: See Ordering Information.

Q: I just bought Sandra Professional. Do I need to register?
A: No, you're automatically registered when you buy the program.

Q: <some> feature does not work!
A: Check out the Standard version thoroughly before ordering. It is WYSIWYG - it's similar for the other modules. Also, modules that don't work/are unavailable do not appear in either version.

Q: Why can't I still register for free for the Professional version 2.xx?
A: There are many various reasons, it's up to you to make up your own mind. Lack of funds, loss of free web & e-mail accounts, almost no donations, almost no help. It was either to try to turn to real shareware or let it die. Since we can always try the latter we've chosen the former.

You can, however, use the Professional version 1.10 using the code ID 7070 (as promised). For the latter 1.xx versions use the code 19248.Check the documentation for more information. Remember this is valid up to (not including) version 2.00 - does not include any later patches.

Q: Why can't I use the old registration code in the Professional version 2.xx?
A: There is no registration code in version 2.xx. The Register Wizard has been removed together with all ancillary resources. So you cannot register this way anymore - the full version and the shareware executables are different.

Q: Is there a code-generator for Sandra?
A: Yes, for version 1.10. It does not work with versions 1.16 or later. Will never work with 2.xx or later since there are no keys to generate. Don't ask for help in finding it, it's illegal. There is no excuse for using it - Standard version gives you around 80% of all modules for free.

Q: Is there a crack for Sandra?
A: No, and we don't think there will be one. The code for the modules is simply not there. There are no nag screens, time-limits or run limits so there's nothing to crack.
