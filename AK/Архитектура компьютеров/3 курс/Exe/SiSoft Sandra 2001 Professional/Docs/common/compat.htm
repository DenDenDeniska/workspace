SiSoft Sandra - The Diagnostic Tool, Compatibility List

This document contains a list of devices on which the latest version of Sandra has been tested successfully. It is not meant to be a complete list since that would be unfeasible. Hardware devices supported indirectly (i.e. through their drivers, Windows itself) are not included.

DISCLAIMER: All information contained herein is for information only, and is provided "as is" with no warranties whatsoever, including any warranty of merchantability, noninfringement, fitness for any particular purpose or any other warranty otherwise arising from it. Please refer to the Sandra Licence included with the product for more information.

Sandra was tested and no problems have been found on the listed devices. However, this does not mean that it will work on similar configurations. It has been tested successfully on at least 15,000+ systems from which reports have been received and catalogued.

NOTE: Due to NDAs we are unable to reveal the full capabilities of our software before the official release date of any such devices. We will add any such information as soon as it becomes publicly available. A device may be fully supported even if it is not mentioned here.
