Updates

Sandra wird regelmäßig upgedated, normalerweise alle 3-4 Monate. Wenn Sie Probleme haben oder neue Features brauchen stellen Sie bitte sicher, ob nicht eine neuere Version verfügbar ist. Benutzten sie den WebUpdate Asisstenten um zu prüfen ob ein Update zur Verfügung steht.
