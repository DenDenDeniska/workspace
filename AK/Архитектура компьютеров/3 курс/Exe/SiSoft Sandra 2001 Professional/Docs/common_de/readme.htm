Installation

Bitte starten Sie Setup.exe und folgen sie den Anweisungen.

Deinstallation des Programms

Öffnen Sie Systemsteuerung - Software und klicken Sie auf SiSoft Sandra. Klicken Sie den Hinzufügen/Entfernen Button und folgen Sie den Anweisungen.

Installer configuration file

setup.lid

Installer configuration file

Sollte eine der Dateien fehlen oder beschädigt sein, versuchen Sie das Programm erneut downzuloaden.

Alle Programmdateien sind gepackt, sie sollten es also dem Install Wizard überlassen die Dateien zu entpacken und auf die Festplatte zu kopieren. Das Installationsprogramm erstellt auch eine Programmgruppe in ihrem Startmenü. Das benutzte Installationsprogramm InstallShield ist das Programm für Windows Anwendungen und wird von den meisten Programmen zur Installation benutzt.

So teilen Sie die Dateien auf, daß sie auf einer (oder mehreren) 1.44MB Diskette(n) Platz finden.

Da das gesamte Programm nicht auf eine 1.44MB Diskette paßt, sind die Dateien so gestaltet, daß sie auf mehrere Disketten verteilt werden können. Um mehrere 1.44MB Installations-Disketten zu erstellen, kopieren sie die Dateien wie folgt:

  • Diskette 2 - data2.cab auf die eine Diskette.
  • Diskette 1 - alle anderen Dateien. Die Dateien sollten auf einer Diskette Platz finden.

Konfigurationsdatei

Und im Windows\System Ordner:

mfc42.dll

Microsoft VC++ MFC Bibliothek

msvcrt.dll

Microsoft VC++ MFC Bibliothek

Diese Dateien werden von den meisten MFC Anwendungen benutzt die mit Microsoft Visual C++ 5.X geschrieben wurden, z. B. die meisten modernen Windows Anwendungen. Es ist deshalb sehr wahrscheinlich, daß sie sich bereits auf ihrem System befinden. In diesem Fall werden sie nicht kopiert.
