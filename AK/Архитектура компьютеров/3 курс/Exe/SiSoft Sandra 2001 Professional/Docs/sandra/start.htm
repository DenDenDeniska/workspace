Installing the Program

Please run Setup.exe and follow the instructions on the screen.

All program files are compressed, so you should let the Install Wizard decompress & copy them to the hard disk. The Install program will also create a program group and add the program items. The InstallShield is the installer for Windows applications and is used by most commercial and shareware applications. It is arguably the best so you are very likely to have it already installed on your machine.

Un-Installing/De-Installing the Program

Go to Control Panel - Add/Remove Programs and choose SiSoft Sandra. Click on Add/Remove button and follow the instructions.

WARNING !!!

IT IS STRONGLY RECOMMENDED THAT YOU SAVE/CLOSE EVERYTHING YOU ARE WORKING AT (open documents, files, etc.) BEFORE RUNNING THE PROGRAM FOR THE FIRST TIME. THE PROGRAM WAS TESTED FOR INCOMPATIBILITIES BUT YOU NEVER KNOW…

If you find no problems then you shound be OK.

Technical Support

Please consult the Questions & Answers document, the help file and the Sandra Web Page before calling for technical support. These documents contain many tips, fixes and answers to common questions. It is very likely that your question was answered already.

Updates & Fixes

Sandra is regularly updated, usually every 3-4 months. If you have problems or need new features, then make sure you check whether a new version is available. Use the WebUpdate wizard to check for a patch and download it.
