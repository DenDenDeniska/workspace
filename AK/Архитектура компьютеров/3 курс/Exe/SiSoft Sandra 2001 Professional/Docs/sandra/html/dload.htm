How to update Sandra

To update Sandra, use the Web-Update Wizard to check for updates that are downloaded and installed automatically (if any are found). You don't need to search for files, decompress and install, everything is done for you automatically. See Help for more information about this functionality.

The image on the left shows you how the module's icon looks like. The module within Sandra herself!

If you cannot find it, go to Help - WebUpdate Wizard menu in Sandra and run it that way.

If you're experiencing problems after the upgrade, refer to Q & A - Install Problems and follow the directions given.
