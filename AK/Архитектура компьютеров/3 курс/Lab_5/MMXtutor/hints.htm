Правила игры

Вам надо предсказать, какой результат получится при выполнении MMX-команд вычитания: psubw, psubusw и psubsw.

Заполните 16-разрядное слово, в котором вы сейчас видите "????". Для этого щелчком мыши выберите один из вариантов ответа, приведенных слева внизу, и нажмите кнопку ОТВЕТ. Если Вам требуется помощь, щелкните мышью на имени команды - на экране появится подсказка.

В любой момент Вы можете начать игру с начала.

Подсказка

Команда psubw MM0,MM2 вычитает слова регистра MM2 из слов регистра MM0 в циклической арифметике.

Обычное вычитание дает:
0001h - 0002h = -1
8000h - 0001h = 32767
7F38h - E8F9h = -27073
0001h - FFFFh = -65534
Числа справа записаны в десятичной системе. Вам осталось применить к ним правила циклической арифметики. Щелчком мыши выберите один из ответов слева внизу.

Подсказка

Команда psubusw MM0,MM2 вычитает слова без знака, находящиеся в регистре MM2, из слов без знака в регистре MM0. Используется арифметика с насыщением. Допустимый диапазон для слов без знака: 0..65535.

Обычное вычитание дает:
0001h - 0002h < 0
8000h - 0001h = 32767
7F38h - E8F9h < 0
0001h - FFFFh < 0
Числа справа записаны в десятичной системе. Вам осталось применить к ним правила арифметики с насыщением. Щелчком мыши выберите один из ответов слева внизу.

Подсказка

Команда psubsw MM0,MM2 вычитает слова со знаком, находящиеся в регистре MM2, из слов со знаком, находящихся в регистре MM0. Используется арифметика с насыщением. Допустимый диапазон для слов со знаком -32768..32767.

Обычное вычитание дает:
0001h - 0002h = -1
8000h - 0001h < -32768
7F38h - E8F9h > 32767
0001h - FFFFh = 2
Числа справа записаны в десятичной системе. Вам осталось применить к ним правила арифметики с насыщением. Щелчком мыши выберите один из ответов слева внизу.
